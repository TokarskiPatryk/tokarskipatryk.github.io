I support companies in Poland and across the EU in data engineering work, from fast audits to production delivery.

## Scope
- Data system audits
- Ad-hoc consulting and technical advisory
- Designing data system architecture
- Implementing ETL/ELT pipelines
- Data modelling

## Preferred collaboration model
- Hourly consulting
- Part-time engagement
- B2B project work
- Remote work (PL/UE) and occasional on-site (PL/UE)

## Typical kickoff

1. Short context from your side: technologies used, data sources, and core problems.
2. Short technical call to clarify goals.
3. Concrete proposal for scope and next steps.
